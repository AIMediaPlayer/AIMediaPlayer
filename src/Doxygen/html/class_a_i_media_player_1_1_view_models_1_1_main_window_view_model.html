Detailed Description

Reprezintă ViewModel-ul principal al aplicației media player, responsabil pentru gestionarea comenzilor și a datelor utilizate de interfața grafică principală.

Clasa implementează logica specifică modelului arhitectural MVVM (Model-View-ViewModel), facilitând comunicarea dintre interfața utilizator și serviciile aplicației.
