Detailed Description

Definește comportamentele specifice fiecărei stări a player-ului media conform șablonului de proiectare State.

Interfața permite separarea logicii de control a player-ului în funcție de starea curentă (ex.: Play, Pause, Stop), oferind implementări diferite pentru acțiunile disponibile utilizatorului.
