Introducere

AIMediaPlayer este o aplicatie multimedia realizata in C#, destinata redarii fisierelor audio si video. Interfata aplicatiei este conceputa astfel incat sa poata fi utilizata usor de catre orice utilizator, fara cunostinte de programare.

Aplicatia ofera functii de baza pentru redarea fisierelor media, gestionarea playlist-urilor si controlul redarii.
